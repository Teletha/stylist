auto()Text

Text

The browser chooses the appropriate offset for underlines.

fromFont()Text

Text

If the font file includes information about a preferred offset, use that value. If the font file doesn't include this information, behave as if auto was set, with the browser choosing an appropriate offset.

length(int length, Unit unit)Text

int length
Unit unit
Text

Specifies the offset of underlines as a , overriding the font file suggestion and the browser default. It is recommended to use em units so the offset scales with the font size.
