inset()Shadow

Shadow

The presence of the inset keyword changes the shadow to one inside the frame (as if the content was depressed inside the box). Inset shadows are drawn inside the border (even transparent ones), above the background, but below content.

offset(intoffsetX, intoffsetY, Unitunit)Shadow

intoffsetX
intoffsetY
Unitunit
Shadow

These are two values to set the shadow offset. offset-x> specifies the horizontal distance. Negative values place the shadow to the left of the element. offset-y> specifies the vertical distance. Negative values place the shadow above the element. See for possible units. If both values are 0, the shadow is placed behind the element (and may generate a blur effect if blur-radius> and/or spread-radius> is set).

blurRadius(doublesize, Unitunit)Shadow

doublesize
Unitunit
Shadow

This is a third value. The larger this value, the bigger the blur, so the shadow becomes bigger and lighter. Negative values are not allowed. If not specified, it will be 0 (the shadow's edge is sharp).

blurRadius(Numericsize)Shadow

Shadow

spreadRadius(doublesize, Unitunit)Shadow

doublesize
Unitunit
Shadow

This is a fourth value. Positive values will cause the shadow to expand and grow bigger, negative values will cause the shadow to shrink. If not specified, it will be 0 (the shadow will be the same size as the element).
