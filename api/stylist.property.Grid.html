The CSS align-items property sets the align-self value on all direct children as a group. In flexbox, it controls the alignment of items on the cross axis. In grid layout, it controls the alignment of items on the block axis within their grid areas.

The CSS place-items shorthand property aligns items along both the block and inline directions at once. It sets the values of the align-items and justify-items properties. If the second value is not set, the first value is also used for it.

flowRow()Grid

Grid

Chainable API

The grid-auto-flow CSS property controls how the auto-placement algorithm works, specifying exactly how auto-placed items get flowed into the grid.

Items are placed by filling each row in turn, adding new rows as necessary. If neither row nor column is provided, row is assumed.

Grid

Chainable API

The grid-auto-flow CSS property controls how the auto-placement algorithm works, specifying exactly how auto-placed items get flowed into the grid.

Items are placed by filling each column in turn, adding new columns as necessary.

flowDense()Grid

Grid

Chainable API

The grid-auto-flow CSS property controls how the auto-placement algorithm works, specifying exactly how auto-placed items get flowed into the grid.

&quot;dense&quot; packing algorithm attempts to fill in holes earlier in the grid, if smaller items come up later. This may cause items to appear out-of-order, when doing so would fill in holes left by larger items.

If it is omitted, a &quot;sparse&quot; algorithm is used, where the placement algorithm only ever moves &quot;forward&quot; in the grid when placing items, never backtracking to fill holes. This ensures that all of the auto-placed items appear &quot;in order&quot;, even if this leaves holes that could have been filled by later items.
